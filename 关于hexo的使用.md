### Leo 简述

### 初始化
```
hexo init
```

### 装依赖
```
yarn
```
or
```
npm install
```

#### 新建文章
```
hexo new '文章标题'
```

### 生成网页文件
```
hexo g
```

### 生成网页文件(清缓存)
```
hexo g -d
```

### 本地服务器预览效果
```
hexo s
```

### 发布
```
hexo d
```
